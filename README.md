#LawangSewu_SVM_and_MultinomialNaiveBayes
This repository hosts a sentiment analysis project centered around Lawang Sewu Semarang, a renowned tourist attraction. The project aims to gauge sentiment expressed in online feedback and reviews regarding Lawang Sewu by employing two distinct machine learning approaches: Support Vector Machine (SVM) and Multinomial Naive Bayes.

#Dataset
We have curated a dataset comprising comments and reviews sourced from diverse platforms including social media, travel forums, and online review portals. These comments encapsulate the viewpoints and experiences of visitors to Lawang Sewu Semarang.

#Methods
Support Vector Machine (SVM): SVM is a supervised learning algorithm utilized for classification tasks. Within this project, SVM is utilized to categorize comments into positive, negative, or neutral sentiments. Model fine-tuning is performed to optimize sentiment classification performance. 
Multinomial Naive Bayes: Multinomial Naive Bayes is a probabilistic machine learning algorithm particularly effective for text classification tasks. We employ Multinomial Naive Bayes to classify comments into sentiment categories and contrast its performance against SVM.

#Key project outcomes include:
Sentiment classification of comments as either positive or negative. Generation of visualizations such as word clouds to illustrate the most prevalent words associated with positive and negative sentiments.
